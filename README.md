# Combat-Tracker_v2

A web app for tracking combat in DnD 5e.

## Requirements
- Flask
- Python

## Use
- ```.\venv\Scripts\activate``` to activate virtual env.
- ```deactivate``` to deactivate virtual env.
- ```py -m flask --debug run``` or ```py app.py```
